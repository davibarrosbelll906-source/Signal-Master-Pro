Ideia NOVA e ÚNICA: Luna Shadow Validation (não existe em lugar nenhum)
Conceito:
Em vez de mostrar o sinal imediatamente, o motor gera o sinal normal e depois cria uma “sombra” (shadow) — uma validação em tempo real feita pela Luna com visão + análise profunda.
Só o sinal que passar pela Luna Shadow é mostrado ao usuário.
Isso é diferente de tudo que existe hoje:

Outras plataformas mostram o sinal e pronto.
Aqui o sinal só aparece se a Luna (com visão) confirmar que o gráfico realmente apoia o sinal.

Como funciona na prática (fluxo):

Motor clássico gera um sinal (score 78, EURUSD CALL).
Backend tira um screenshot automático do gráfico atual (lightweight-charts).
Envia a imagem + dados do sinal para a Luna.
Luna responde com:
Aprovação (Sim / Não)
Confidence (0–100)
Explicação curta (para mostrar ao usuário)

Se Luna aprovar com confidence ≥ 75 → sinal é liberado.
Se rejeitar → sinal é descartado silenciosamente e o usuário nunca vê.

Vantagem brutal:

Assertividade sobe muito porque a IA está vendo o gráfico real (não só números).
O usuário recebe só sinais de altíssima qualidade + explicação educativa da Luna.


Estrutura Pronta para Implementar (Luna Shadow Validation)
1. Nova função no signalEngine.ts (backend)
TypeScriptprivate async validateWithLunaShadow(signal: any, chartCanvas: HTMLCanvasElement) {
  const base64Image = chartCanvas.toDataURL('image/png');

  const result = await lunaAgent.invoke({
    messages: [{
      role: "user",
      content: [
        { type: "text", text: `
          Você é Luna Shadow Validator.
          Analise se este sinal está realmente bom no gráfico atual.
          Par: ${signal.pair} | Direção: ${signal.direction} | Score: ${signal.score}
          Só aprove se realmente houver confluência clara.
          Responda apenas em JSON:
          { "approved": true/false, "confidence": 0-100, "shortExplanation": "..." }
        `},
        { type: "image_url", image_url: { url: base64Image } }
      ]
    }]
  });

  const validation = JSON.parse(result.content);
  return validation;
}
2. No fluxo de geração de sinal (dentro do generateSignals):
TypeScriptconst rawSignal = { ... }; // sinal gerado normalmente

// Validação Shadow
const shadowValidation = await this.validateWithLunaShadow(rawSignal, currentChartCanvas);

if (!shadowValidation.approved || shadowValidation.confidence < 75) {
  return; // sinal descartado silenciosamente
}

// Se passou na validação, libera o sinal
const finalSignal = {
  ...rawSignal,
  lunaShadowConfidence: shadowValidation.confidence,
  lunaExplanation: shadowValidation.shortExplanation
};

this.io.emit('new_signal', finalSignal);
