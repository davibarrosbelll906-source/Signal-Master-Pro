# 🤖 PROMPT COMPLETO — AI NEXUS STUDIO (para Replit / qualquer IA)

---

## INSTRUÇÃO PRINCIPAL

Crie um aplicativo web completo chamado **AI Nexus Studio** em um único arquivo HTML autocontido (`index.html`). Não use frameworks, não use React, não use Node.js. Tudo em **HTML5 + CSS3 + JavaScript ES6 puro**, sem dependências externas além das listadas abaixo.

---

### Implementação das APIs

#### Claude (Anthropic) — SSE Streaming
```javascript
fetch('https://api.anthropic.com/v1/messages', {
  method: 'POST',
  headers: {
    'x-api-key': KEY,
    'anthropic-version': '2023-06-01',
    'anthropic-dangerous-direct-browser-access': 'true',
    'content-type': 'application/json'
  },
  body: JSON.stringify({
    model: modelName,
    max_tokens: maxTokens,
    stream: true,
    system: systemPrompt,
    messages: history
  })
})
// Parse SSE: event:content_block_delta → delta.text
```

#### OpenAI — SSE Streaming
```javascript
fetch('https://api.openai.com/v1/chat/completions', {
  headers: { 'Authorization': `Bearer ${KEY}`, 'Content-Type': 'application/json' },
  body: JSON.stringify({ model, stream: true, messages })
})
// Parse SSE: data: {"choices":[{"delta":{"content":"..."}}]}
```

#### OpenRouter — NON-STREAMING (modelos free não suportam SSE)
```javascript
fetch('https://openrouter.ai/api/v1/chat/completions', {
```

### Funcionalidades do Chat
1. **Upload de imagem** (📷) — vision, aceita múltiplas imagens, converte para base64
2. **Upload de documento** (📄) — PDF (via PDF.js), TXT, MD, CSV — extrai texto e injeta no contexto
3. **Microfone** (🎤) — Web Speech API, transcrição em tempo real
4. **TTS** (🔊) — leitura automática das respostas (toggle). Prioridade: ElevenLabs → Web Speech API
5. **Memória** (🧠) — modal para adicionar/remover fatos persistentes (localStorage)
6. **Favoritos** (⭐) — salva prompts favoritos, dropdown com lista
7. **Busca Web** (🌐) — toggle "sempre buscar" (azul = ativo), atalho Ctrl+W
8. **Drag & Drop** — arrastar imagens direto na janela do chat

### Busca Web em Tempo Real (v2.0)
```javascript
// Hierarquia de fontes (tenta na ordem):
// 1. Serper.dev (Google real-time) — 2500/mês grátis
// 2. Brave Search — 2000/mês grátis
// 3. DuckDuckGo Instant Answers — sem key, sempre grátis
// 4. Wikipedia — fallback final

// Toggle "sempre buscar":
let webSearchAlways = localStorage.getItem('web_search_always') === '1';
// Botão 🌐 na toolbar → toggleWebSearch()

// Auto-detecção por palavras-chave:
const triggers = [
  /\b(hoje|agora|atual|recente|últim[ao]|notícia|news|novidade|lançamento)\b/i,
  /\b(202[4-9]|203\d)\b/,
  /\b(preço|cotação|valor|quanto|bolsa|crypto|bitcoin|btc|dólar)\b/i,
  /\b(clima|temperatura|previsão do tempo)\b/i,
  /\b(gpt-[4-9]|claude [3-9]|gemini|llama|deepseek|openai|anthropic)\b/i,
  // ... etc
];

// Serper.dev API:
fetch('https://google.serper.dev/search', {
  method: 'POST',
  headers: { 'X-API-KEY': serperKey, 'Content-Type': 'application/json' },
  body: JSON.stringify({ q: query, num: 6, hl: 'pt', gl: 'br' })
})
// Retorna: answerBox, knowledgeGraph, news, organic results

// DuckDuckGo (sem key):
fetch(`https://api.duckduckgo.com/?q=${query}&format=json&no_html=1&skip_disambig=1`)
```

## PÁGINA 3: VÍDEOS

### API fal.ai — Kling AI
```javascript
// 1. Submeter job
const submit = await fetch('https://queue.fal.run/fal-ai/kling-video/v1/standard/text-to-video', {
  method: 'POST',
  headers: { 'Authorization': `Key ${FAL_KEY}`, 'Content-Type': 'application/json' },
  body: JSON.stringify({ prompt, duration: '5', aspect_ratio: '16:9' })
});
const { request_id } = await submit.json();

// 2. Polling a cada 5 segundos
const poll = await fetch(`https://queue.fal.run/fal-ai/kling-video/v1/standard/text-to-video/requests/${request_id}`, {
  headers: { 'Authorization': `Key ${FAL_KEY}` }
});
// status: 'IN_PROGRESS' | 'COMPLETED'
// quando COMPLETED: output.video.url
```

### UI
- Sidebar com: textarea do prompt, duração (5s/10s), proporção (16:9, 9:16, 1:1)
- Card de loading com barra animada (3-5 minutos de espera)
- Player de vídeo quando pronto

```javascript
function autoSaveSession() {
  const sessions = JSON.parse(localStorage.getItem('chat_sessions') || '[]');
  const session = {
    id: Date.now(),
    title: chatHistory[0]?.content?.slice(0,50) || 'Nova conversa',
    msgs: chatHistory,
    date: new Date().toISOString(),
    model: currentModel,
    tokens: sessionTokens
  };
  sessions.unshift(session);
  if (sessions.length > 50) sessions.pop(); // max 50 sessões
  localStorage.setItem('chat_sessions', JSON.stringify(sessions));
}
```
