Aqui está a estrutura completa e pronta para adicionar a ideia que você mais gostou:

Biblioteca de Análises Salvas com Luna
(“Minhas Análises com Luna”)
Essa feature permite que toda análise de gráfico (foto + resposta da Luna) seja salva automaticamente. O usuário pode revisar lições antigas, buscar por par, data ou conceito (ex: “suporte e resistência”), e aprender de forma organizada.
1. Estrutura de Dados (localStorage por enquanto)
Adicione esta nova chave no seu sistema de localStorage:
TypeScript// Nova chave
smpLunaAnalyses7 → Array de análises salvas
Exemplo de um item salvo:
TypeScript{
  id: "ana_1744060800000",           // timestamp como ID
  date: "2026-04-07 14:35",
  pair: "EURUSD",
  timeframe: "M5",
  imageBase64: "data:image/png;base64,...",   // opcional (pode ser pesado)
  userQuestion: "Analise este gráfico e me explique a tendência",
  lunaResponse: "Este gráfico mostra uma clara tendência de alta com... (resposta completa da Luna)",
  keyLessons: ["Suporte em 1.0820", "Pin bar de reversão", "Confluência com sessão London"],
  score: 87,                         // score do sinal na hora da análise (opcional)
  tags: ["tendencia", "suporte_resistencia", "price_action"]
}

  func: async (data) => {

Na rota /api/luna/chat, depois de receber a resposta da Luna, você pode automaticamente chamar a tool de salvar (ou fazer isso no frontend).
3. Nova Página no Frontend
Crie a página:
src/pages/LunaAnalysesPage.tsx
Estrutura básica da página:
tsximport { useState, useEffect } from 'react';
import { format } from 'date-fns'; // ou sua lib de data

export default function LunaAnalysesPage() {
  const [analyses, setAnalyses] = useState([]);
  const [search, setSearch] = useState('');
  const [filterPair, setFilterPair] = useState('Todos');

  useEffect(() => {
    const saved = JSON.parse(localStorage.getItem('smpLunaAnalyses7') || '[]');
    setAnalyses(saved);
  }, []);

  const filteredAnalyses = analyses
    .filter(a => 
      (filterPair === 'Todos' || a.pair === filterPair) &&
      (a.pair.toLowerCase().includes(search.toLowerCase()) || 
       a.lunaResponse.toLowerCase().includes(search.toLowerCase()))
    )
    .sort((a, b) => b.id - a.id); // mais recentes primeiro

  return (
    <div className="p-6">
      <h1 className="text-3xl font-bold mb-6">📚 Minhas Análises com Luna</h1>
      
      {/* Filtros */}
      <div className="flex gap-4 mb-6">
        <input
          type="text"
          placeholder="Buscar por par ou conceito..."
          value={search}
          onChange={(e) => setSearch(e.target.value)}
          className="glass-card flex-1 px-4 py-3"
        />
        <select value={filterPair} onChange={(e) => setFilterPair(e.target.value)} className="glass-card px-4 py-3">
          <option value="Todos">Todos os pares</option>
          {/* Preencha com pares conhecidos */}
        </select>
      </div>

      <div className="grid grid-cols-1 md:grid-cols-2 lg:grid-cols-3 gap-6">
        {filteredAnalyses.map((analysis) => (
          <div key={analysis.id} className="glass-card p-5 rounded-2xl">
            <div className="flex justify-between text-sm text-gray-400 mb-3">
              <span>{analysis.date}</span>
              <span className="font-medium">{analysis.pair} • {analysis.timeframe}</span>
            </div>

            {analysis.imageBase64 && (
              <img 
                src={analysis.imageBase64} 
                alt="Gráfico analisado" 
                className="w-full rounded-lg mb-4 border border-gray-700"
              />
            )}

            <p className="text-sm text-violet-300 mb-3 line-clamp-3">
              {analysis.userQuestion}
            </p>

            <div className="text-sm leading-relaxed mb-4">
              {analysis.lunaResponse.substring(0, 220)}...
            </div>

            <div className="flex flex-wrap gap-2">
              {analysis.keyLessons?.map((lesson, i) => (
                <span key={i} className="text-xs bg-green-900/50 text-green-400 px-3 py-1 rounded-full">
                  {lesson}
                </span>
              ))}
            </div>

            <button className="mt-4 text-violet-400 hover:text-violet-300 text-sm">
              Ver análise completa →
            </button>
          </div>
        ))}
      </div>

      {filteredAnalyses.length === 0 && (
        <p className="text-center text-gray-500 mt-12">Nenhuma análise salva ainda. Tire uma foto do gráfico e peça para Luna analisar!</p>
      )}
    </div>
  );
}
Adicione essa página na Sidebar (Sidebar.tsx) no grupo “Aprendizado” ou “Ferramentas”.
4. Como salvar automaticamente (no momento da análise)
No componente de captura (LunaChartAnalyzer.tsx), após receber a resposta da Luna:
TypeScript// Após receber a resposta
const newAnalysis = {
  id: Date.now().toString(),
  date: new Date().toLocaleString('pt-BR'),
  pair,
  timeframe,
  imageBase64: base64Image,        // opcional – pode remover se ocupar muito espaço
  userQuestion: message,
  lunaResponse: reply,
  keyLessons: [],                  // Luna pode retornar uma lista de lições
  tags: ["price_action", "tendencia"]
};

// Salvar
const existing = JSON.parse(localStorage.getItem('smpLunaAnalyses7') || '[]');
localStorage.setItem('smpLunaAnalyses7', JSON.stringify([newAnalysis, ...existing]));
